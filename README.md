# Portfólio Pessoal - Ana Beatriz

Bem-vindo ao repositório do meu portfólio pessoal! Este projeto foi desenvolvido para apresentar minhas habilidades como Desenvolvedora Front-end, exibir meus projetos recentes e facilitar o contato para oportunidades profissionais, especialmente na busca por um estágio em Engenharia de Software.

🌍 **[Acesse o projeto online aqui!](https://portfolioanabeatrizdev.netlify.app/)**

## Visão Geral

O site foi construído com foco em **UI/UX**, utilizando um design *Soft Minimalism*, garantindo uma navegação fluida, responsiva e agradável em qualquer dispositivo.

### Funcionalidades
* **Design Responsivo:** Adaptável para desktops, tablets e smartphones utilizando Media Queries.
* **Animações Suaves:** Efeitos de rolagem revelando os elementos gradativamente.
* **Navegação Dinâmica:** Menu mobile interativo (Hamburger Menu) e rolagem suave entre as seções.
* **Galeria de Projetos:** Cards interativos com links diretos para o código-fonte e para a aplicação em produção.

## Tecnologias Utilizadas

Este projeto foi construído do zero, sem a utilização de frameworks pesados, para demonstrar domínio nos fundamentos da web:

* **HTML5:** Estruturação semântica e acessível.
* **CSS3:** Estilização moderna utilizando Custom Properties (Variáveis), Flexbox e CSS Grid.
* **JavaScript (Vanilla):** Lógica para o funcionamento do menu mobile e configuração das animações.
* **Remix Icon:** Biblioteca de ícones vetoriais.
* **ScrollReveal.js:** Biblioteca JavaScript para as animações de entrada ao rolar a página.
